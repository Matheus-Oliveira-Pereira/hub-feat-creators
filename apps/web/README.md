# Web — HUB Feat Creators

Next.js 14 (App Router) + React 18 + TypeScript strict + Tailwind.

## Setup
```
pnpm install
```

## Comandos
- `pnpm dev` — dev server (porta 3000)
- `pnpm build` — produção
- `pnpm lint` — ESLint (next/core-web-vitals)
- `pnpm typecheck` — tsc strict
- `pnpm test` — Vitest
- `pnpm test:e2e` — Playwright (após `pnpm exec playwright install`)
